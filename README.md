# AI-Powered Retail Business Intelligence & Decision Support System

## Problem Statement
Retail businesses generate large volumes of transactional data but often struggle to convert it into actionable insights.
This project builds an end-to-end business intelligence and analytics system to analyze retail transaction data, track KPIs,
identify customer behavior patterns, predict churn, and forecast future sales to support data-driven decision-making.

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL (MySQL)
- Power BI
- Machine Learning (Forecasting, Classification)

## Project Structure
- data/        : Datasets (raw & processed)
- notebooks/   : Data cleaning and analysis
- sql/         : SQL queries and schema
- dashboards/  : BI dashboards
- models/      : ML models
